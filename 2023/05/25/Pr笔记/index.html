Pr笔记

发表于2023-05-25|更新于2023-05-25|Pr笔记
|字数总计:2.3k|阅读时长:6分钟|阅读量:

多选素材

多选一列

​ 点击第一个图片，按住shift点击最后一张图片，可以把首尾之间图片进行选择

单个图片多选

​ 按住alt单击需要选择的图片，可以进行图片单个多选

工作区域放大

选中 pr 中要放大的面板，点击键盘~可以进行放大缩小

调整时间轴

手动调节

​ 按住alt + 滚轮可以进行手动调节

自动调节

​ 点击键盘| 键，自动调节时间轴合适大小，注意要在英文状态下

V1与A1

总的V1与A1

​ V1指的是视频

​ A1指的是音频

​ 点亮后可以插入视频或音频，不点亮则不能插入

小的V1,V2,V3…和A1,A2,A3…

​ 点亮后，粘贴优先粘贴在当前点亮轨道，

​ 多个点亮，则按从小到大优先级点亮

批量添加转场效果

  1. 点击你所要添加的转场效果
  2. 右键点击设为默认过度
  3. 鼠标左键划取所要添加的部分
  4. 点击键盘Ctrl + D进行添加

批量修改转场时间

  1. 找到上方菜单栏里的编辑
  2. 找到里面的首选项
  3. 点击时间轴
  4. 修改默认视频过渡持续时间
  5. 可能需要重新选中所要添加的部分，然后点击键盘Ctrl + D进行添加

修改批量添加的视频持续时间

  1. 选中需要修改的部分
  2. 右击，选择速度/持续时间...（快捷键Ctrl + R）
  3. 修改持续的时间
  4. 记得勾选波纹编辑，移动尾部剪辑，否则的话，各个图片之间，会有间隔

复制素材的方法

  1. Ctrl + C 和 Ctrl + V
  2. 按住Alt把素材往上拖或往下拖

批量处理效果

先做好一个，然后把做好的效果，给其他素材

  1. 选中做好的素材，然后右键点击复制 （快捷键 Ctrl + C）
  2. 选中想要批处理的素材，右击粘贴属性，然后选择所要粘贴的属性（快捷键 Ctrl + Alt + V）

粘贴属性快捷键Ctrl + Alt + V

光晕素材黑色解决

  1. 点击素材，然后点击左上效果控件
  2. 找到不透明度中的混合模式，修改为滤色

剃刀工具裁剪多段素材

使用剃刀工具时，按住shift 进行裁剪，可以多段素材裁剪

视频大小和序列大小不相同

视频比序列小，有黑边

  1. 选中序列中的视频
  2. 右键，点击缩放为帧大小

字幕遮罩

用于遮挡原本视频存在的字幕

  1. 在新建序列的地方，选择新建 颜色遮罩
  2. 把颜色遮罩拖到时间轴上
  3. 在 效果里面搜索 裁剪
  4. 把裁剪效果添加到颜色遮上
  5. 在左上效果控件里找到裁剪
  6. 设置顶部 或 底部 大小为 88%

添加字幕或文字

  1. 找到想要添加字幕的地方
  2. 按住Ctrl + T可以添加字幕
  3. 按 T或选择 文本工具 可以进行文本内容编辑

标记

点击M可以对时间轴进行标记

点击M 按住Alt 进行拖动，可以对一段时间进行标记

生成新的独立小视频片段

  1. 选中你裁切好的小视频
  2. 右击，选择 制作子剪辑
  3. 不勾选 将修剪限制为子剪辑边界

视频启用或隐藏

按住shift + E可以启用视频 或者隐藏视频 （音频也同时）

批量把视频后移

使用向前选择工具（快捷键A） （和剃刀工具在一栏）

视频预览小技巧

第一次点击L是两倍速度

第二次点击L是一倍速度

….. 依此递增

点击K是暂停

点击J是倒退

清除时间轴间隙

法一： 单个删除，点击空隙，右键，删除波纹

法二： 批量删除，选中区域，在顶部菜单栏找到序列，选择封闭间隙（注意：只有高版本才有）

法三：在缩短时间时，勾选波纹编辑，移动尾部剪辑

右移5帧

按住 shift + 右键，向右移5帧

制作花字

  1. 点击上方文件，找到 新建 里面的旧版标题
  2. 其他就按需所求
  3. 如果字体在背景下面，就 把字体Ctrl + X 剪切，然后 Ctrl + V粘贴
  4. 花字做好，需要拖到时间轴才能使用

拖动矩形缩小

拖动缩小矩形时，如果想两边同时缩小，拖动时按住Alt 键

关键帧动画

  1. 选择所要添加关键帧的物体，点击左上的 效果控件
  2. 点击 文字前面闹钟 就可以添加关键帧动画
  3. 每换一个时间点，挪一下位置，都会生成关键帧动画
  4. 两个不同效果添加关键帧时，跨可以按住 shift 可以自动吸附到下一个点，或者是点击 转到下一关键帧

关键帧动画效果

​ 位置变化

​ 缩放

​ 旋转

​ 透明度

蒙版

在 效果控件里面有个 不透明度，下面有个椭圆和长方形，点击后创建蒙版

作用：框定一个范围，黑色的不显示，被选中的显示

删除：点选蒙版，然后按Del键

自定义路径时：为方便调整路径可以按住Alt拖动

音频淡入淡出

  1. 选中音频，点击左下效果控件
  2. 选择音频过渡里面的 恒定功率

视频调色

  1. 点击最上方菜单栏里的窗口
  2. 找到窗口里的Lumetri Color / Lumetri 颜色
  3. 在 基本校正里面，进行调节 曝光，对比度，阴影等….

QW法 快速处理废片

Q：选择你要处理的视频，时间轴指针找到对应位置，点击Q 键删除前面的内容

W：选择你要处理的视频，时间轴指针找到对应位置，点击W 键删除后面的内容

追踪动画

给贴图设置追踪动画

  1. 选中贴纸，找到效果控件里面的运动
  2. 点击位置，调整位置
  3. 设置关键帧动画，然后shift + 右键每5帧调整位置
  4. 多次调整位置就是追踪动画

帧定格（视频画面暂停）

添加帧定格，可以使视频画面暂停

  1. 选中素材，在时间轴上选择相应位置
  2. 右键素材，选择添加帧定格
  3. 整个画面就在当前位置定住

视频黑白翻转

视频颜色变成黑白

  1. 点选素材
  2. 找到添加效果界面（左下）
  3. 搜索黑白，把黑白效果拖动到素材上

调整图层

是个空图层，但是可以把效果添加到调整图层

例如，给视频添加黑白效果，可以把效果添加到调整图层，自由调整隔离图层，可以自由调整黑白位置和时间等

在新建序列的地方，可以新建调整图层

设置视频快放和慢放

  1. 选中视频，按R键
  2. 拖动视频，可以快放或着慢放

视频部分位置方大（头部方法效果）

注意： 放大前，先把需要方大的地方，用剃刀工具裁剪开，不然一整段视频都有放大效果

  1. 素材裁开，选中需要放大的区域
  2. 在效果中（左下）搜索 放大（在视频效果 -> 扭曲 -> 放大）
  3. 在效果控件中（左上），通过调整位置，大小，羽化值，设置放大效果
  4. 可以添加羽化值，放大的更柔和；可以给 放大率添加关键帧动画，有个放大过度

视频扣取某个部分

把视频中某个人物或物品抠出来，让物品可以单独运动

  1. 选中视频，找到合适位置，点击视频预览处（右上）的 导出帧 / Ctrl + shift + E
  2. 然后把你抠出的帧，重新导入到时间轴
  3. 在效果控件中（左上），在 不透明度中，选择钢笔工具，使用蒙版把人物或物体进行抠取

如何保留背景？

  1. 把你导出来的帧放到ps里面
  2. 使用 矩形框选工具，框中人物或物品
  3. 然后按键盘 shift +F5内容识别
  4. 注意在英文状态下

更改旋转中心点

​ 在效果控件里面，找到锚点，更改数值，可以改变物体中心点

视频翻转效果

例如视频左右翻转

  1. 选中视频，找到效果（左下）
  2. 找到视频效果->变换
  3. 在里面挑选水平翻转或垂直翻转

视频镜像效果

视频上有多个当前画面

  1. 选中视频，找到效果（左下）
  2. 找到视频效果->风格化->复制
  3. 左上 效果控件 可以修改 计数，改变屏幕上数量

标记入点I 和 标记出点O

标记入点和标记出点可以选择一段片段

选择了一个片段时，可以按Ctrl + M以序列切入/序列切出方式导出当前片段
